[ ] Go to https://www.npmjs.com/signup
[ ] Create account and verify email

STEP 5: Login to npm
[ ] Open terminal in this folder
[ ] Run: npm login
[ ] Enter username and password

STEP 6: Test Publish (Optional)
[ ] Run: npm pack               (creates clickaway-1.0.0.tgz)
[ ] This lets you preview what will be published
